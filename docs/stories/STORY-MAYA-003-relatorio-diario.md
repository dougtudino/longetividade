  Manhã (6-12): "Bom dia [nome]!"
  Tarde (12-18): "Boa tarde [nome]!"
  Noite (18-6): "Boa noite [nome]!"

- [ ] AC-03: Tom feminino, animado, focado em resultados
  Exemplos de mensagens iniciais que Maya pode usar

- [ ] AC-04: Link rápido para pendências do checklist de setup
  Se setup incompleto: "Você tem X itens pendentes para configurar"

- [ ] AC-05: Histórico de conversa salvo em MayaMessage
  Últimas 20 mensagens carregadas ao abrir o chat
